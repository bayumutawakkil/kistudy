Part V: Infrastructure Security

Applied cryptography, networks, hardware, cyber-physical systems, and telecommunications

17Applied Cryptography

Overview

Practical application of cryptography including implementation considerations, key management, and deployment of cryptographic systems in real-world scenarios.

Key Topics
  • Cryptographic Libraries: OpenSSL, libsodium, implementation bugs
  • Key Management: Key generation, storage, derivation, rotation
  • Public Key Infrastructure (PKI): Certificate authorities, trust anchors
  • Digital Certificates: X.509, validation, revocation (CRL, OCSP)
  • TLS/SSL Deployment: Configuration, cipher suites, protocol versions
  • Side-Channel Attacks: Timing attacks, power analysis, cache attacks
  • Randomness & Entropy: Random number generation, seeding
  • Key Escrow & Backdoors: Policy and technical implications
Source: CyBOK v1.1.0 - Applied Cryptography. Crown Copyright 2021.

18Network Security

Overview

Security of network infrastructure from physical layer to application layer, covering protocols, threats, detection, and defense mechanisms.

Key Topics
  • Network Protocols: TCP/IP stack, routing, DNS, BGP
  • Protocol Vulnerabilities: Spoofing, sniffing, replay attacks
  • DDoS Attacks: Volumetric, protocol-based, application-layer attacks
  • Man-in-the-Middle (MitM): ARP poisoning, DNS hijacking, SSL stripping
  • Firewall & Filtering: Stateful inspection, packet filtering, proxy firewalls
  • Intrusion Detection/Prevention: IDS/IPS, anomaly detection
  • VPN & Tunneling: IPsec, GRE, secure remote access
  • Software-Defined Networking (SDN): Security implications
  • Network Monitoring: Flow analysis, NETFLOW, packet capture
Source: CyBOK v1.1.0 - Network Security. Crown Copyright 2021.

19Hardware Security

Overview

Physical security of computing devices including secure processors, tamper resistance, and protection against physical attacks and chip-level threats.

Key Topics
  • Trusted Platform Module (TPM): Root of trust, attestation, key storage
  • Secure Boot: UEFi secure boot, chain of trust, firmware security
  • Hardware Attacks: Side-channel analysis, fault injection, physical tampering
  • Power Analysis: Simple and differential power analysis
  • Timing Attacks: Cache attacks (Spectre, Meltdown), branch prediction
  • Hardware Trojans: Malicious modifications in chip design
  • Cryptographic Accelerators: Specialized hardware implementation
  • Physically Unclonable Functions (PUF): Unique device identifiers
  • Secure Enclaves: Intel SGX, ARM TrustZone security
Source: CyBOK v1.1.0 - Hardware Security. Crown Copyright 2021.

20Cyber-Physical Systems Security

Overview

Security of systems that combine computation with physical world interaction, where cyber attacks can have direct physical consequences.

Key Topics
  • SCADA & Industrial Control: Legacy systems, real-time constraints
  • Critical Infrastructure: Power grids, water systems, transportation
  • IoT Security: Constrained devices, wireless networks, firmware updates
  • Autonomous Vehicles: Vehicle-to-everything (V2X), sensor security
  • Medical Devices: Implantable devices, hospital systems, interoperability
  • Robotics: Physical safety constraints, human-robot interaction
  • Anomaly Detection: Real-time threat detection in operational systems
  • Resilience & Redundancy: Fault tolerance, failsafe mechanisms
  • Compliance & Standards: IEC 62443, NIST guidelines for CPS
Source: CyBOK v1.1.0 - Cyber-Physical Systems Security. Crown Copyright 2021.

21Physical Layer & Telecommunications Security

Overview

Security at the physical layer of wireless and wired communications, addressing electromagnetic and physical threats to communication channels.

Key Topics
  • Spectrum & Electromagnetic Security: Jamming, eavesdropping, interference
  • Physical Layer Cryptography: Exploiting channel characteristics for security
  • Wireless Security: Wi-Fi, Bluetooth, cellular (4G/5G) protocols
  • Anti-Jamming & Frequency Hopping: Resilience to interference
  • Device Fingerprinting: Identifying devices from physical layer signatures
  • Secure Ranging & Positioning: GPS spoofing, distance verification
  • Covert Channels: Side-channel information leakage
  • TEMPEST & Emanations Security: Unintentional electromagnetic leakage
  • Optical & Quantum Communications: Emerging secure communication methods
Source: CyBOK v1.1.0 - Physical Layer & Telecommunications Security. Crown Copyright 2021.
