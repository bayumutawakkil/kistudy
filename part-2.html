Part II: Attacks & Defences

Threats, attack technologies, incident management, and forensics

5Malware & Attack Technologies

Overview

Comprehensive study of malicious software including viruses, worms, Trojans, ransomware, and spyware. Covers development techniques, detection methods, and the malware ecosystem.

Key Topics
  • Malware Types: Viruses, worms, Trojans, ransomware, spyware, adware, rootkits
  • Obfuscation & Evasion: Polymorphism, metamorphism, code encryption, anti-analysis
  • Propagation Methods: Network worms, email, drive-by downloads, software vulnerabilities
  • Malware Ecosystem: Command & control infrastructure, botnet operations
  • Detection Techniques: Signature-based detection, heuristic analysis, behavioral detection
  • Analysis & Reverse Engineering: Static and dynamic malware analysis
  • Mitigation & Defense: Host-based and network-based detection systems
Source: CyBOK v1.1.0 - Malware & Attack Technologies. Crown Copyright 2021.

6Adversarial Behaviours

Overview

Understanding cyber attackers - from individual threat actors to organized criminal groups and nation-state actors. Maps attack methodologies and motivations.

Key Topics
  • Threat Actor Classification: Script kiddies, hacktivists, organized crime, nation-states
  • Motivations: Financial gain, political objectives, espionage, disruption
  • Attack Frameworks: MITRE ATT&CK, Cyber Kill Chain, attack trees
  • Advanced Persistent Threats (APT): Long-term, targeted campaigns
  • Threat Intelligence: Indicators of Compromise (IOCs), threat attribution
  • Attack Infrastructure: Botnets, C2 servers, privacy tools used by attackers
  • Adversary Simulation: Red teaming, penetration testing
Source: CyBOK v1.1.0 - Adversarial Behaviours. Crown Copyright 2021.

7Security Operations & Incident Management

Overview

Day-to-day security operations including detection, investigation, response, and recovery from security incidents.

Key Topics
  • Security Operations Centers (SOC): Organization, staffing, tools
  • Detection Systems: Intrusion Detection Systems (IDS), IPS, firewalls
  • SIEM Platforms: Security Information & Event Management tools
  • Log Analysis & Correlation: Finding patterns in security events
  • Threat Hunting: Proactive threat discovery
  • Incident Response Process: Detection, containment, eradication, recovery
  • Post-Incident Activities: Lessons learned, improvements
Source: CyBOK v1.1.0 - Security Operations & Incident Management. Crown Copyright 2021.

8Forensics

Overview

Digital forensics - the science of collecting, preserving, and analyzing digital evidence for legal proceedings.

Key Topics
  • Forensic Process: Identification, preservation, analysis, reporting
  • Chain of Custody: Maintaining evidence integrity for legal admissibility
  • Disk Forensics: File system analysis, data recovery, timeline reconstruction
  • Memory Forensics: RAM analysis, volatile data capture
  • Network Forensics: Packet analysis, network traffic reconstruction
  • Mobile Device Forensics: Android and iOS evidence extraction
  • Cloud Forensics: Challenges in cloud environments, shared responsibility
  • Encryption & Data Destruction: Challenges when data is encrypted
Source: CyBOK v1.1.0 - Forensics. Crown Copyright 2021.
