Bagian III: Arsitektur Keamanan & Daur Hidup

Kriptografi, keamanan sistem, metode formal, dan kontrol akses

9Kriptografi

Ikhtisar

Fondasi matematis kriptografi modern. Mencakup enkripsi simetris dan asimetris, fungsi hash, tanda tangan digital, dan kriptografi pasca-kuantum.

Topik Utama
  • Enkripsi Simetris: AES, mode cipher blok, cipher stream
  • Enkripsi Asimetris: RSA, Elliptic Curve Cryptography (ECC), transportasi kunci
  • Fungsi Hash: Keluarga SHA-x, properti kriptografis
  • Tanda Tangan Digital: Penandatanganan dan verifikasi, otentikasi kunci
  • Protokol Pertukaran Kunci: Diffie-Hellman, ECDH, kerahasiaan ke depan sempurna
  • Definisi Keamanan Formal: Keamanan semantik, indistinguishability
  • Kriptanalisis: Memecahkan skema kriptografi, serangan side-channel
  • Kriptografi Pasca-Kuantum: Skema tahan terhadap komputer kuantum
Sumber: CyBOK v1.1.0 - Kriptografi. Hak Cipta Mahkota 2021.

10Sistem Operasi & Virtualisasi

Ikhtisar

Keamanan di level OS dan hypervisor. Mencakup isolasi proses, kontrol akses wajib, eksekusi terpercaya, dan keamanan mesin virtual.

Topik Utama
  • Isolasi Proses: Perlindungan memori, layout ruang alamat
  • Model Kontrol Akses: Kontrol akses diskresioner (DAC) dan wajib (MAC)
  • Sistem Berbasis Kemampuan: Model kontrol akses alternatif
  • Keamanan Kernel: Tingkat privilege, validasi system call
  • Lingkungan Eksekusi Terpercaya: TEE, ARM TrustZone, Intel SGX
  • Keamanan VM: Isolasi hypervisor, serangan lintas-VM
  • Secure Boot & UEFI: Verifikasi integritas boot
Sumber: CyBOK v1.1.0 - Sistem Operasi & Virtualisasi. Hak Cipta Mahkota 2021.

11Keamanan Sistem Terdistribusi

Ikhtisar

Keamanan dalam arsitektur terdistribusi termasuk cloud, sistem peer-to-peer, dan blockchain. Mengatasi kepercayaan, konsensus, dan integritas data di seluruh node otonom.

Topik Utama
  • Toleransi Kesalahan Bizantium: Perjanjian di lingkungan berbahaya
  • Mekanisme Konsensus: Proof-of-Work, Proof-of-Stake, perjanjian Bizantium
  • Keamanan Blockchain: Serangan 51%, kerentanan smart contract
  • Keamanan Cloud: Multi-tenancy, model tanggung jawab bersama
  • Microservices: Otentikasi layanan-ke-layanan, keamanan API
  • Model Kepercayaan: Arsitektur zero-trust, batasan kepercayaan
  • Konsistensi Data: Teorema CAP, konsistensi akhirnya
Sumber: CyBOK v1.1.0 - Keamanan Sistem Terdistribusi. Hak Cipta Mahkota 2021.

12Metode Formal untuk Keamanan

Ikhtisar

Penggunaan matematika dan logika untuk secara formal membuktikan sifat keamanan sistem. Mencakup model checking, theorem proving, dan verifikasi protokol.

Topik Utama
  • Spesifikasi Formal: Mesin keadaan, logika temporal
  • Model Checking: Verifikasi otomatis dari sistem terbatas
  • Theorem Proving: Pembukti teorema interaktif dan otomatis
  • Analisis Protokol: ProVerif, Tamarin, penegakan kebijakan keamanan
  • Model Antagonis: Definisi formal dari kemampuan penyerang
  • Properti Keamanan: Kerahasiaan, otentikasi, non-repudiasi
  • Aplikasi: Verifikasi TLS, analisis protokol kriptografis
Sumber: CyBOK v1.1.0 - Metode Formal untuk Keamanan. Hak Cipta Mahkota 2021.

13Otentikasi, Otorisasi & Akuntabilitas (AAA)

Ikhtisar

Tiga pilar kontrol akses: membuktikan identitas, memberikan izin, dan mencatat tindakan. Penting untuk keamanan di setiap sistem.

Topik Utama
  • Metode Otentikasi: Kata sandi, biometrik, token perangkat keras, MFA
  • Model Otorisasi: RBAC, ABAC, ACL, daftar kemampuan
  • Kerangka Kontrol Akses: XACML, mesin kebijakan
  • Single Sign-On (SSO): Federasi, SAML, OpenID Connect, OAuth
  • Manajemen Identitas: Penyediaan pengguna, manajemen siklus hidup
  • Akuntabilitas & Audit: Logging, jejak audit, analisis forensik
  • Non-Repudiasi: Mencegah penyangkalan tindakan, tanda tangan digital
Sumber: CyBOK v1.1.0 - Otentikasi, Otorisasi & Akuntabilitas. Hak Cipta Mahkota 2021.
