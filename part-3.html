Part III: Security Architecture & Lifecycle

Cryptography, systems security, formal methods, and access control

9Cryptography

Overview

Mathematical foundations of modern cryptography. Covers symmetric and asymmetric encryption, hash functions, digital signatures, and post-quantum cryptography.

Key Topics
  • Symmetric Encryption: AES, block cipher modes, stream ciphers
  • Asymmetric Encryption: RSA, Elliptic Curve Cryptography (ECC), key transport
  • Hash Functions: SHA-x family, cryptographic properties
  • Digital Signatures: Signing and verification, key authentication
  • Key Exchange Protocols: Diffie-Hellman, ECDH, perfect forward secrecy
  • Formal Security Definitions: Semantic security, indistinguishability
  • Cryptanalysis: Breaking cryptographic schemes, side-channel attacks
  • Post-Quantum Cryptography: Schemes resistant to quantum computers
Source: CyBOK v1.1.0 - Cryptography. Crown Copyright 2021.

10Operating Systems & Virtualisation

Overview

Security at the OS and hypervisor level. Covers process isolation, mandatory access control, trusted execution, and virtual machine security.

Key Topics
  • Process Isolation: Memory protection, address space layout
  • Access Control Models: Discretionary (DAC) and Mandatory (MAC) access control
  • Capability-Based Systems: Alternative access control models
  • Kernel Security: Privilege levels, system call validation
  • Trusted Execution Environments: TEE, ARM TrustZone, Intel SGX
  • VM Security: Hypervisor isolation, cross-VM attacks
  • Secure Boot & UEFI: Boot integrity verification
Source: CyBOK v1.1.0 - Operating Systems & Virtualisation. Crown Copyright 2021.

11Distributed Systems Security

Overview

Security in distributed architectures including cloud, peer-to-peer systems, and blockchain. Addresses trust, consensus, and data integrity across autonomous nodes.

Key Topics
  • Byzantine Fault Tolerance: Agreement in malicious environments
  • Consensus Mechanisms: Proof-of-Work, Proof-of-Stake, Byzantine agreements
  • Blockchain Security: 51% attacks, smart contract vulnerabilities
  • Cloud Security: Multi-tenancy, shared responsibility model
  • Microservices: Service-to-service authentication, API security
  • Trust Models: Zero-trust architecture, trust boundaries
  • Data Consistency: CAP theorem, eventual consistency
Source: CyBOK v1.1.0 - Distributed Systems Security. Crown Copyright 2021.

12Formal Methods for Security

Overview

Use of mathematics and logic to formally prove security properties of systems. Includes model checking, theorem proving, and protocol verification.

Key Topics
  • Formal Specifications: State machines, temporal logic
  • Model Checking: Automated verification of finite systems
  • Theorem Proving: Interactive and automated theorem provers
  • Protocol Analysis: ProVerif, Tamarin, security policy enforcement
  • Adversary Models: Formal definition of attacker capabilities
  • Security Properties: Secrecy, authentication, non-repudiation
  • Applications: TLS verification, cryptographic protocol analysis
Source: CyBOK v1.1.0 - Formal Methods for Security. Crown Copyright 2021.

13Authentication, Authorisation & Accountability (AAA)

Overview

The three pillars of access control: proving identity, granting permissions, and recording actions. Critical for security in every system.

Key Topics
  • Authentication Methods: Passwords, biometrics, hardware tokens, MFA
  • Authorization Models: RBAC, ABAC, ACLs, capability lists
  • Access Control Frameworks: XACML, policy engines
  • Single Sign-On (SSO): Federation, SAML, OpenID Connect, OAuth
  • Identity Management: User provisioning, lifecycle management
  • Accountability & Audit: Logging, audit trails, forensic analysis
  • Non-Repudiation: Preventing denial of actions, digital signatures
Source: CyBOK v1.1.0 - Authentication, Authorisation & Accountability. Crown Copyright 2021.
