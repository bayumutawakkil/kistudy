Part IV: Software & Platform Security

Secure development, web/mobile vulnerabilities, and software lifecycle

14Software Security

Overview

Implementation-level security vulnerabilities in software and techniques to prevent and mitigate them. Covers memory safety issues, injection flaws, and runtime protections.

Key Topics
  • Memory Safety Vulnerabilities: Buffer overflows, use-after-free, heap corruption
  • Injection Flaws: SQL injection, command injection, code injection
  • Race Conditions: Timing vulnerabilities, TOCTOU (Time-of-check-time-of-use)
  • Logic Flaws: Broken authentication, authorization bypass
  • Static Analysis: Source code scanning, SAST tools
  • Dynamic Testing: Fuzzing, symbolic execution, runtime monitoring
  • Runtime Protections: ASLR, DEP/NX, stack canaries, Control Flow Guard
  • Secure Coding Practices: Input validation, output encoding, secure libraries
Source: CyBOK v1.1.0 - Software Security. Crown Copyright 2021.

15Web & Mobile Security

Overview

Security challenges specific to web applications and mobile platforms, covering client-side and server-side vulnerabilities.

Key Topics
  • Web Client Vulnerabilities: Cross-Site Scripting (XSS), clickjacking, phishing
  • Web Server Vulnerabilities: SQL injection, CSRF, insecure deserialization
  • Browser Security: Same-Origin Policy (SOP), Content Security Policy (CSP)
  • HTTPS & TLS: Certificate validation, pinning, mixed content
  • Authentication: OAuth, OpenID Connect, WebAuthn, password managers
  • Session Management: Cookies, tokens, session hijacking
  • Mobile Platforms: iOS and Android security models, app permissions
  • Mobile Threats: Malware, reverse engineering, data exfiltration
  • APIs & Microservices: API security, rate limiting, authentication
Source: CyBOK v1.1.0 - Web & Mobile Security. Crown Copyright 2021.

16Secure Software Lifecycle

Overview

Integrating security throughout the software development lifecycle from design through deployment and maintenance.

Key Topics
  • Lifecycle Models: Microsoft SDL, SAFECode, Touchpoints framework
  • Secure Design: Threat modeling, architecture review, security patterns
  • Secure Development: Secure coding standards, code review, pair programming
  • Testing & Verification: Penetration testing, security testing, CI/CD integration
  • Agile & DevSecOps: Continuous security in rapid development
  • Third-Party Risk: Dependency management, supply chain security
  • Deployment & Maintenance: Secure configuration, patching, updates
  • Maturity Models: SAMM, BSIMM, Common Criteria
  • Domain-Specific: Cloud, IoT, embedded systems, autonomous vehicles
Source: CyBOK v1.1.0 - Secure Software Lifecycle. Crown Copyright 2021.
